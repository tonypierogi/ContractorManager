# TimeTrack Pro - Contractor Timesheet & Invoice Tool

A modern web application for managing contractor timesheets and generating invoices, built with Supabase backend.

## Features

### For Contractors/Employees
- **Time Clock**: Clock in/out with real-time tracking
- **Manual Entries**: Add shifts manually if needed
- **View Shifts**: See all time entries with filtering by date
- **Copy to Spreadsheet**: Export data in tab-separated format for easy pasting into Excel/Google Sheets
- **Profile Management**: Update personal info, phone, billing address
- **View Invoices**: See invoices generated by the employer

### For Employers/Admins
- **Team Overview**: See all team members and their hourly rates
- **Set Hourly Rates**: Configure rates for each contractor
- **View All Timesheets**: Filter by employee and date range
- **Generate Invoices**: Create professional invoices for any employee and period
- **Business Settings**: Configure company details and payment instructions
- **Copy to Spreadsheet**: Export timesheet data for accounting

## Setup Instructions

### 1. Create a Supabase Project

1. Go to [supabase.com](https://supabase.com) and create a free account
2. Create a new project
3. Wait for the project to be provisioned

### 2. Set Up the Database

1. In your Supabase dashboard, go to **SQL Editor**
2. Copy the contents of `supabase-schema.sql` from this project
3. Paste and run the SQL to create all tables and security policies

### 3. Configure the App

1. In Supabase, go to **Settings** → **API**
2. Copy your **Project URL** and **anon/public key**
3. Edit `config.js` and replace the placeholder values:

```javascript
const SUPABASE_CONFIG = {
    url: 'https://your-project-id.supabase.co',  // Your Project URL
    anonKey: 'your-anon-key-here'                 // Your anon/public key
};
```

### 4. Create an Admin User

1. Open the app and sign up with your email
2. In Supabase, go to **Table Editor** → **profiles**
3. Find your user and change the `role` from `employee` to `admin`
4. Refresh the app - you'll now see the admin dashboard

### 5. Run the App

Simply open `index.html` in a web browser, or serve it with any HTTP server:

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx serve

# Using PHP
php -S localhost:8080
```

Then open `http://localhost:8080` in your browser.

## Usage Guide

### As an Admin

1. **First time setup**: Go to Business Settings and enter your company details
2. **Add team members**: Share the app URL - contractors sign up themselves
3. **Set rates**: Click on a team member to set their hourly rate
4. **View timesheets**: Go to Timesheets tab to see all logged hours
5. **Generate invoices**: Go to Invoices, select employee and date range, generate

### As a Contractor

1. **Sign up**: Create account with your email
2. **Update profile**: Add your billing address and phone number
3. **Clock in/out**: Use the Time Clock to track your work
4. **Add manual entries**: Use "Add Manual Entry" for any missed clock-ins
5. **View your hours**: Check My Shifts to see logged time
6. **Export data**: Use "Copy to Spreadsheet" to get your data

## Data Export Format

The "Copy to Spreadsheet" feature exports data in tab-separated format:

```
Date    Clock In    Clock Out    Hours    Description    Rate    Amount
Jan 15, 2026    09:00 AM    05:00 PM    8.00    Development work    75    $600.00
```

Simply paste this into Excel, Google Sheets, or any spreadsheet app.

## Security

- Row Level Security (RLS) ensures users can only see their own data
- Admins can view and manage all data
- All data is stored securely in Supabase
- Authentication is handled by Supabase Auth

## Customization

### Change Colors
Edit the CSS variables in `styles.css`:

```css
:root {
    --accent-primary: #00d4aa;    /* Main accent color */
    --accent-secondary: #00b894;  /* Secondary accent */
    --bg-primary: #0a0f1a;        /* Background color */
    /* ... */
}
```

### Add Tax/Additional Fields
Modify the invoice generation in `app.js` and update the `invoices` table in Supabase.
